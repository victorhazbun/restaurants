# README

## Ruby version

2.5.3

## System dependencies

- PostgreSQL

## Configuration

- `cp .env.sample .env`
- Update the `.env` file with corresponding values
- `bundle install`

## Database creation

- `rails db:create`

##  Database initialization

- `rails db:migrate`
- `rails db:seed`

## How to test with postman

- Download the Postman collection https://www.getpostman.com/collections/37cb4a4342cb06a63facf
